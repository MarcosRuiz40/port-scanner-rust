# port-scanner-rust
Escáner de puertos concurrente en Rust usando TcpStream, threads y timeout.

## Ejecución rápida
```
cargo run 
```
## Funcionalidades
* Busca los puertos abierto en la red
* Menciona que puertos estan abiertos actualmente

# Ejemplo real 
```
Puerto 135 abierto
Puerto 445 abierto
```

* Uso de TcpStream
* Uso de Thread

* Uso de concurrencia al hacer 2 hilos
* Uso de Timeout
